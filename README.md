# DIRECT

Data Integration Run-time Execution Control Tool

This repository contains the following:

* Data Model (DDL and DML)
* Management tool (C#)
* SSIS examples with Biml code generation for DIRECT hooks
* T-SQL hooks
* Functional specification
* Support SQL scripts
